# Task

| 方法 | HTTP | 路径 |
| --- | --- | --- |
| `create` | POST | `/tasks` |
| `update` | PUT | `/tasks/:id` |
| `get` | GET | `/tasks/:id` |
| `listByExecution` | GET | `/executions/:executionId/tasks` |
| `start` | POST | `/tasks/:id/start` |
| `finish` | POST | `/tasks/:id/finish` |
| `close` | POST | `/tasks/:id/close` |
| `activate` | POST | `/tasks/:id/active` |
| `delete` | DELETE | `/tasks/:id` |
